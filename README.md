# advanced-data-structures-s2025
Материалы семинара «Современные структуры данных»: порядок выступлений, файлы статей, краткие содержания докладов.

## О курсе

Преподаватель — [Б. Золотов](https://t.me/boris_a_z).

Продолжительность курса — 15 пар, каждый вторник с 17 февраля по 26 мая, 19:00, ауд. 102.

Студенты выпускных курсов заканчивают раньше и сдают зачёт только по докладам, случившимся к этому моменту.

## Порядок получения зачёта

Для получения зачёта необходимо:
- сделать в рамках семинара доклад по предложенной или согласованной теме,
- написать _summary_ / краткий, но исчерпывающий пересказ собственного доклада,
- на зачёте — ответить на несколько вопросов по докладам семинара, пересказать основные алгоритмы или сформулировать теоремы.
